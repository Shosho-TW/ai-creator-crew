# AI Creator Crew｜張修修的 AI 船員市集

《AI 創作者新世紀》課程的 plugin 市集。**每個 plugin 是一位 AI 船員**——你是總編輯，船員交草稿、你做判斷。

## 安裝（一次加入，之後自動更新）

**Claude Cowork**：Customize → Plugins → **Add marketplace** → 貼上本 repo 網址 → 選要僱用的船員按 Install。
**Claude Code**：`/plugin marketplace add <owner>/ai-creator-crew` → `/plugin install librarian@ai-creator-crew`

## 船員名冊

| Plugin | 船員 | 一句 JD | 內含 skills |
|---|---|---|---|
| [librarian](plugins/librarian/) | 圖書館員 | 管好你的第二大腦，人寫卡、AI 跑腿 | librarian-setup、capture、ingest、daily-review、kb-search、centaur-writer |
| [content-strategist](plugins/content-strategist/) | 內容企劃 | 讓你永遠有下一支能做的題 | scout（資訊採集雷達）、topic-picker（主題挑選器） |

更多船員（社群編輯、包裝編輯、數據分析師⋯）陸續上船。
